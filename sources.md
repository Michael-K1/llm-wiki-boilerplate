# Research Sources

Curated list of sources for wiki-researcher. Organize by reliability tier.
The agent searches these sources first, respecting tier priority.

## How to Use

Add websites, databases, and portals relevant to your vault's domain.
Each entry follows the format:

```
- [Source Name](https://url) — Description | keywords: topic1, topic2
```

**Keywords** help the agent match your sources to search queries.
**Tiers** control search priority — Tier 1 sources are searched first.
